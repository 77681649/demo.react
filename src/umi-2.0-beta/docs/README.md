### 入门非常简单

```bash
# 安装
$ yarn global add umi # 或者：npm install -g umi

# 新建一个页面
$ mkdir pages
$ echo 'export default () => <h1>Hello umi</h1>;' > pages/index.js

# 启动调试服务器
$ umi dev

# 构建
$ umi build
```
